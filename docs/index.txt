#######################################
dsigma: galaxy-galaxy lensing made easy
#######################################

.. image:: plot.png
   :width: 100 %
   :align: center

:code:`dsigma` is an easy-to-use pipeline for analyzing galaxy-galaxy lensing.
The package is written in python, has a broadly applicable API and is
optimized for computational efficiency. While originally intended to be used
with the shape catalog of the Hyper-Suprime Cam (HSC) survey, it is also works
with surveys like the Canada-France-Hawaii Telescope Lensing Survey (CFHTLenS),
the Kilo-Degree Survey (KiDS) or the Dark Energy Survey (DES).
